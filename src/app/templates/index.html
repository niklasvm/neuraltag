NeuralTag is an application that automatically names your activities using your past naming conventions as well as some creativity

How does it work:

  • Start by connecting your Strava account - leave the permissions as is in order to allow the app to work correctly
  • Create a new activity with strava
  • After your activity is saved, NeuralTag will automatically name your activity and append "named with NeuralTag ðŸ¤–" to the description
  • NeuralTag will learn from your naming conventions and improve over time
  • If you don't like the name you can set your activity to Rename and NeuralTag will try again
  • If you still don't like the name you can just change it
  • If you don't want NeuralTag to rename your activites anymore you can opt out by navigating to My Apps and select revoke access for Neuraltag
